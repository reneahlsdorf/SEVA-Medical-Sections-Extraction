## Support Vector Classifier

The models are hosted as a part of the PhysioNet SEVA publication. **Place the `model_<X>.parquet` files here for model loading in the Jupyter notebook** 
The Jupyter notebook `notebooks/load_models.ipynb` contains exemplary code how to load the model parquet files.

> ⚠️ Please make sure that the correct packages (see `requirements.txt`) are installed to be able to load the models successfully
